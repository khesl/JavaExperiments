Программа курса
Без «воды» и лекций для галочки
Верстка. Основы
Узнайте, что происходит «под капотом» сайта, что такое backend и frontend. Как выглядит и из чего состоит код веб-страницы. Устанавливаем программы для разработки, изучаем Axure для прототипирования проекта перед началом непосредственно кодинга.

Практика: Установить себе на компьютер редактор Sublime Text. Открыть код веб-страницы index.html, внести в него правки и опубликовать результат.
Основа JavaScript
Добавляем верстке интерактивности. Узнайте, что такое JavaScript, и как он подключается к веб-странице. Научитесь подключать и пользоваться библиотекой jQuery, которая упрощает жизнь разработчика и помогает легко получать доступ к любому элементу кода. Узнайте о фреймворках JavaScript, которые позволяют структурировать код, чтобы он был легко поддерживаемым.

Практика: На странице новостного агентства, под новостями, вывести курс рубля к доллару и к евро через ресурс fixer.io. Дизайн – на ваше усмотрение, но выглядеть должно аккуратно, в стиле заданного макета сайта.
Объектно-ориентированное программирование
Освойте самую распространенную и востребованную парадигму, которая используется во всех современных языках. Применяйте на практике принципы объектно-ориентированного программирования: абстракцию, инкапсуляцию, наследование, полиморфизм. Пользуйтесь классами, объектами, конструкторами, виртуальными функциями.

Практика: Создайте иерархию сущностей (конструкторов, из которых можно создать экземпляры объектов):
• базовый конструктор «Транспортное средство» с общими для всех транспортных средств свойствами и методами
• дочерние конструкторы: автомобиль, самолёт, корабль, которые наследуют общие свойства и методы от родительского
• продемонстрируйте, как дочерние сущности могут переопределять родительские свойства и методы в соответствии со своим собственным поведением


React.js
Пользуйтесь готовой библиотекой для создания пользовательских интерфейсов. Вы должны использовать React, так как он отлично подходит для командной разработки, UI код читабельный и прост в сопровождении. Смотрите в будущее веб-разработки.

Практика: Напишите виджет комментариев на React:
1. Отображение списка комментариев: автор, текст, дата и время
2. Добавление нового комментария (поля для ввода: имя автора, текст)
3. Удаление комментария (кнопка удаления рядом с каждым комментарием)
4. Сохранение состояния приложения в localStorage в формате JSON.
Результат выложите на GitHub или пришлите в виде архива.


База данных MySQL
Изучите, как установить и запустить MySQL сервер для разработки, как управлять пользователями баз данных. Разработаете структуру базы данных MySQL для вашего приложения с нуля, и узнайте, как импортировать и экспортировать данные.

Практика: спроектировать базу для хранения информации о пользователях, с возможностью разделения пользователей по группам, чтобы им можно было давать разные права к определенным разделам сайта.
PHP – язык внутренней кухни сайта
Делайте сайты, которые и доносят информацию и поражают воображение, и имеют четко отлаженный механизмом. PHP создан для разработки внутренней, серверной части сайта. Чёткая обработка загружаемых файлов, операции с базами данных, проверка авторизации пользователей, анализ cookies ― без этих элементов сегодня не представить серьёзный веб-проект.
